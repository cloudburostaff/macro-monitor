# Macro Monitor

Automated macroeconomic surveillance — a weekly executive briefing on inflation, the labour market, growth, housing, sentiment, central-bank policy, and financial conditions.

**Live site:** https://cloudburostaff.github.io/macro-monitor/

A swarm of seven specialist monitoring agents tracks primary-source data releases as they land; a central **Macro Strategist** synthesises them into a single read of the economic cycle and publishes a five-minute briefing here every Saturday.

## Structure

- `docs/index.html` — landing page + weekly report archive
- `docs/reports/YYYY-Www.html` — individual weekly editions (ISO week)

Published automatically via the Macro Monitor swarm. _Internal monitoring summary — not investment advice._
